# LLM 生成广告图维度划分文献

排序：选择 2021-2026 年、UTD24 商学院期刊；其次选择 CCF-A 会议中与 text-to-image/生成图评价直接相关的论文；最后补充少量心理学顶刊或 2020 年高相关经典文献。  
用途：支撑把 LLM/API/YoYo 链路生成的广告图拆成可实验操控的维度，例如图文一致性、广告框架、主体呈现、视觉质量、语义对齐、品牌对齐、AI 披露与真实感。

## 推荐维度

| 维度 | 可操控方式 | 主要支撑文献 |
|---|---|---|
| 生成来源与披露 | AI disclosed vs undisclosed；API 生成 vs YoYo 工作流生成 | #1, #2, #3, #10, #20 |
| 图文关系 | 图文一致 vs 不一致；文字多 vs 图片主导；文字位置/覆盖面积 | #4, #5, #9, #14 |
| 广告框架 | informational/功能型 vs emotional/lifestyle 型；产品卖点 vs 使用场景 | #2, #8, #12, #14 |
| 主体呈现 | packshot/纯商品图 vs 手持/使用场景/人物出镜 | #6, #11, #13, #20 |
| 视觉质量 | aesthetics、detail quality、colorfulness、professional-looking、复杂度 | #7, #15, #16, #18, #19 |
| 语义对齐 | prompt-image alignment；商品、颜色、材质、数量、空间关系是否准确 | #16, #17, #18, #19 |
| 品牌对齐 | brand-compatible vs only aesthetically pleasing；品牌风格一致性 | #1, #2, #13 |

## 给本文研究设计的落地建议

可以把 YoYo/LLM 生成广告图的实验操控写成一个 2 x 2 或 2 x 2 x 2 设计：

| 操控维度 | 低/基准条件 | 高/处理条件 | 主要文献 |
|---|---|---|---|
| 广告框架 | 功能型/信息型：商品清楚、卖点模块化、浅色电商主图 | 情绪型/生活方式：真实场景、氛围、人物或使用线索 | #2, #8, #12 |
| 图文关系 | 图文弱一致或仅商品标题 | 图文高度一致；标题、场景、产品功能共同指向同一卖点 | #4, #5, #9 |
| 视觉复杂度 | 极简背景、少物体、低色彩复杂度 | 丰富背景、多物体、较高色彩/场景复杂度 | #12, #14, #15 |
| 语义对齐 | prompt 与图片部分偏离，材质/场景/空间关系不稳定 | 商品、材质、颜色、空间关系均与 prompt 对齐 | #15, #16 |
| AI 披露 | 不披露 AI 生成 | 明确披露 AI-generated / AI-edited | #10 |

更贴近 我们自己的 链路的变量写法：

- `background style`：生活方式、信息型电商、极简棚拍、奢华布景、科技极简、植物自然。
- `product scale / whitespace`：商品占画面比例、中心性、留白比例。
- `template overlay`：文字面积、价格/促销语位置、图文一致性。
- `prompt-image alignment`：商品类别、颜色、材质、数量、空间关系、场景是否匹配。
- `AI disclosure`：广告图旁是否标注 AI 生成或 AI 辅助生成。
