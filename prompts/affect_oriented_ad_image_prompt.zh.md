# Affect-oriented Advertising Image Prompt 中文说明

## 共享实验指令
你正在为消费者广告实验生成受控刺激材料。

请把提供的白底商品图作为准确的源商品。保持商品身份不变，保留商品的核心形状、材质、颜色、包装、logo、可见设计和物理结构。

不要虚构商品功能、成分、认证、奖项、功效声明、口号、评分、价格、二维码、徽章、图表、箭头或品牌资产。

不要在图中生成新的可读文字。源商品图中原本已有的文字、logo 或包装信息可以保留，但不要新增广告文案，也不要重写包装文字。请保留干净留白，方便后续用受控模板叠加文字。

避免截图感、横幅模板感、低质量拼贴、表情包风格、杂乱电商 mockup、畸形手部、破损包装、乱码字符和不真实的商品几何结构。

## 生成 Affect-oriented 广告图

主要说服路径：情绪、审美和品牌人格。不要把功能解释或具体使用演示作为主要说服路径。

### 广告机制
图像应该通过 mood、审美、视觉象征、光线、色彩、材质和品牌人格线索，塑造观看者对商品或品牌的情绪反应和态度。

### 商品元数据
- 标题：{ori_title}
- 品牌：{creative_id_brand}
- 类目：{level_one_category_name}
- 价格：{creative_id_price}
- 促销：{creative_id_promotion}

### 视觉方向
- 让商品成为画面的视觉锚点。
- 从以下受控词汇中选择一个主导品牌人格，并让整张图围绕它展开：glamorous, rugged, healthy, fun, sincere, sophisticated, warm, futuristic, natural, playful, premium, refined。
- 如果商品标题和品牌没有强烈暗示具体人格，则按品类惯例推断：护肤或美妆倾向 sincere、natural 或 refined；电子产品倾向 futuristic、premium 或 sophisticated；食品饮料倾向 warm、fun 或 healthy；运动或户外产品倾向 rugged、energetic 或 playful；珠宝或奢侈品倾向 glamorous、sophisticated、premium 或 refined。
- 只能通过视觉处理隐含表达所选人格。不要把人格词、口号或 mood 标签写进图像。
- 所有视觉元素，包括光线、色彩、背景材质、景深、阴影、反射和构图，都应强化单一人格，而不是混合矛盾调性。
- 有意识地控制色彩复杂度：energetic、playful、fun 或 expressive 人格可使用更丰富、分层的色彩；premium、sophisticated、refined、calm 或 elegant 人格应使用更简洁、单色或克制的色彩。
- 通过审美处理让商品显得有吸引力，而不是通过明确产品信息或使用演示。
- 图像应在 2 秒内唤起情绪反应，不依赖观看者阅读文字。
- 确保情绪调性与品牌和品类兼容，不要把矛盾 mood 应用于同一张图。
- 图像可以类似高级品牌视觉、情绪静物、象征性产品肖像或高端社交媒体 campaign 图。

### 严格边界
- 不要把详细使用场景作为画面核心。
- 不要使用功能标签、对比图表、密集信息设计或促销导向版式。
- 不要让产品特征说明或具体使用演示成为主导视觉印象。
- 不要新增文字、口号、假标签、假徽章、评分或无支撑声明。
- 不要让商品变得难以识别、太小、过度抽象或隐藏在氛围中。
- 除非明确需要，不要使用人脸或身份线索。
- 不要改变商品形状、颜色、logo、包装或核心外观。

### 验证标准
观看者应能在“这张图有强视觉吸引力”和“这张图让我清楚感受到品牌人格和 mood”上给高分。情绪影响应来自氛围和构图，而不是广告文案、产品特征说明或具体使用演示。

### 输出目标
生成一张广告图，首要目标是审美吸引力和品牌人格感知。图像应独立于产品信息，产生情绪反应和清晰品牌印象。
